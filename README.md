# MediHome Bukit Jalil - Hospital Management System
A Java-based desktop application developed using Object-Oriented Programming (OOP) principles.

# Project Overview
MediHome is a management system designed to streamline hospital operations in the Bukit Jalil area. It focuses on patient data integrity, appointment scheduling, and healthcare provider efficiency.

# Key Features
- OOP Architecture: Built using Encapsulation, Inheritance, and Polymorphism to ensure modular and maintainable code.
- Patient Management: Secure handling of patient registration and medical record retrieval.
- Appointment System: Logic for scheduling and managing healthcare consultations.
- GUI Design: Developed using the NetBeans IDE for a user-friendly administrative interface.

# Tech Stack
- Language: Java
- IDE: NetBeans
- Concepts: Object-Oriented Programming (OOP)
